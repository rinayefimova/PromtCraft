# PromptCraft

PromptCraft is a web-based text adventure game where players learn how to communicate with AI by solving puzzles using prompts.

In this game, the world is controlled by artificial intelligence, 
and players must use clear and effective prompts to progress through different challenges.

## Gameplay

- Players type prompts to interact with AI systems
- Each level presents a different challenge
- The game checks if the prompt contains meaningful instructions
- Hints are available if the player gets stuck
- Progression happens when the correct prompt is given
– Convince the final AI to grant access  

---

## Purpose

This project explores how people interact with AI and how small changes in prompts can lead to different results. 
It turns prompt engineering into an interactive and engaging experience.

---

## Features

- Prompt-based gameplay
- Flexible input (not exact commands)
- Hint system
- Level progression
- Restart functionality
- Simple web interface
